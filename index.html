PDF → Excel Converter

Select one or more invoice PDF files. Each is converted to an Excel file (.xlsx) that you can download. Everything happens inside your own browser — no files are uploaded to any server.

This tool recognises invoices from VIKING Life-Saving Equipment, HAY, and Solar. VIKING and HAY convert directly. Solar invoices are read with OCR in your browser (the first Solar file downloads the OCR engine, and OCR takes a few seconds per page); their line totals are checked against the printed invoice total, and any file that doesn't reconcile is flagged for review. Anything unrecognised is skipped with a note, and the rest of your files still process.
