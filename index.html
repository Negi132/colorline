PDF → Excel Converter

Select one or more invoice PDF files. Each is converted to an Excel file (.xlsx) that you can download. Everything happens inside your own browser — no files are uploaded to any server.

Please note: this converter has so far only been tested with PDF invoices from Color Line Cargo. Invoices from other companies may use a different layout and might not convert correctly.
